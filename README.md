# Darklee Theme Set
> ## A set of Visual Studio Code themes that are on the dark side of the moon.<br>
> ### Themes Included
> - #### Deep Dark
> - #### Army

---

<br>

## Installation

1. Open the extensions sidebar on Visual Studio Code
1. Search for Darklee
1. Click Install
1. Select the settings button on the bottom left > Color Theme ï¼ž Choose one of 
    - "Darklee - Army"
    - "Darklee - Deep Dark"
1. Consider rating it in the market place
